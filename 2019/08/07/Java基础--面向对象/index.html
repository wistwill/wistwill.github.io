Posted on 2019-08-07 | In Java |
Words count in article: 1.5k | Reading time ≈ 6

面向过程与面向对象

区别

区别 举例
面向过程 强调的是过程，实现功能，需要一步步去做 自己做饭（一步步去做）
面向对象 强调的是对象，交给对象完成功能 定外卖（下单）

面向对象的3个基本思想特点

符合我们思考问题的一种思想（懒人思想，把工作交给别人（对象）去做）

将复杂的事情简单化（对使用者，对象内部还是很复杂）

将我们从执行者变成了指挥者角色发生了转换

类与对象

类

概念

具有相同属性和行为的对象的集合，是对现实事物的抽象

现实世界 java java
事物（猫） 类（class Cat） 对象（小橘猫）
属性（品种、年龄） 成员变量（breed、age） 成员变量（小橘猫、六个月、黄白相间）
行为（吃鱼、玩毛线球） 成员方法（eat、play） 成员方法（吃鱼、逮老鼠、玩毛线球）

成员变量

和以前定义变量是一样的，只不过位置发生了改变。在类中，方法外。

成员方法

和以前定义方法是一样的，只不过把static去掉，后面在详细讲解static的作用。

有返回类型

基本类型（四类8种）

引用类型（数组、接口…）

无返回类型

用void修饰的（一般与打印相关）

对象

对象是某类事物的具体体现（具有某些独有的属性和行为）

对象的创建与使用

创建对象格式

类名 对象名 = new 类名();

对象存储在堆内存中，有默认值（每创建一个对象（new一下），堆内存都会为其开辟一个空间，并对这个对象的成员变量赋初始值，之后把内存首地址返回给局部变量）

获取成员

获取成员变量：对象名.成员变量名

获取成员方法：对象名.成员方法名

局部变量存储在栈内存通过地址指向堆内存的对象

成员变量与局部变量的区别

区别 成员变量 局部变量
在类中的位置 在类中，方法外 在方法中或者方法声明上(形式参数)
初始值 有默认值 没有默认值，在使用前，必须先定义并赋值
生命周期 随着对象的创建而存在，随着对象的消失而消失 随着方法的调用而存在，随着方法的调用完毕而消失
内存位置 堆内存 栈内存（方法加载在栈，局部变量存储在栈内存通过地址指向堆内存的对象
作用域 整个类 代码块{}

private:

私有的意思，关键字，权限修饰符

可以修饰成员变量，修饰成员方法（成员变量一般用private修饰，成员方法一般为public）

修饰的成员只能在本类中使用

使用private对成员变量进行封装，使用set方法，get方法对成员变量进行间接访问

this：

代表当前所在类（本类）的对象的引用

为了区分成员变量与局部变量重名（变量的使用规则：就近原则）

代表所在类的对象引用，哪一个对象来调用此方法，this就代表哪一个对象
